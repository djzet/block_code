# block_code
## Проект направлен на создание web-приложения для визуальной интепритации визуальных блоков в код
*Стек технологий:* Bootstrap, HTML, CSS, JavaScript

**Что будет реализовано:**

1. Объявление целочисленных переменных
    + Блок для объявления переменных 
    + Возможность объявлять несколько переменных через запятую
    + По умолчанию значение = 0

2. Оператор присваивания
    + Блок для присваивания значения переменной
    + Структура: `переменная = выражение`

3. Базовые арифметические операции
    + Поддержка: `+` `-` `*` `/` `%`
    + В выражениях можно использовать переменные и константы
    + *Опционально:* поддержка приоритета операций и скобок

4. Оператор `If`
    + Блок для условия `>` `<` `==` `!=` `>=` `<=`
    + Вложенный блок для команд, выполняемых при истинном условии

5. Блоки `Begin` / `End`
    + *Опционально*

6. Подсветка ошибок
    + Визуальное выделение блоков с синтаксическими ошибками

#### Категория «Я могу написать сортировку пузырьком»

1. Циклы
    + Реализация цикла `while` или `for` (аналог C++) 
    + Возможность вложенных циклов

2. Оператор `If-Else`
    + Расширенный блок с ветками `if` и `else`

3. Статические массивы
    + Объявление массивов с указанием размера
    + Обращение к элементам по индексу
    + Возможность использования в выражениях

4. Логические выражения
    + Поддержка операторов `AND` `OR` `NOT`
    + Возможность использования скобок

5. Демонстрация работы

#### Категория «HARD» (опционально, на выбор)
1. Функции (объявление, вызов, возврат значения)
2. Сложные выражения в условиях и индексах массивов
3. Строковый и символьный типы данных
4. Вещественные числа
5. Конвертация типов
6. Dataflow-подобный интерфейс
7. Сохранение/загрузка алгоритмов
8. Расширенные структуры данных (матрицы, ООП, лямбды) 
9. Режим отладки

## Ограничения и запреты

+ **Нельзя** использовать внешние интерпретаторы (eval, JS-движки)

+ **Нельзя** конвертировать блоки в текст и исполнять как код JS

+ Вся логика интерпретации должна быть реализована вручную

+ **Нельзя** копировать чужой код без указания авторства и понимания логики

+ Использование генеративных моделей (ChatGPT) для написания основного кода запрещено
